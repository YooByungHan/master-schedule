## ✨ 주요 기능

### 공정표 작성
- **CPM 공정표** — 계획(P) / 실적(A) 이중 바 차트
- **Total Float 시각화** — 꺾인 기준선(Progress Line)
- **선후행 연결선** — 선행/후행 연결 화살표
- **대분류 / 중분류 / 하위 공종** — 무제한 계층 구조
- **물량 · 단위 · 금액 · 실적율** — 수량 관리 통합 (금액·이벤트 열은 기본 접힘, 버튼으로 표시)
- **이벤트 노드** — 마름모 / 빨강마름모 / 원 도형 이벤트 표시
- **일 / 주(5일) / 10일 단위** 차트 전환, 기간 범위 설정, 공휴일 반영, 그레이(흑백) 모드
- **PDF / PNG / Excel 출력**

### AI 분석
- 🔍 **지연 공종 탐지** — 완료 지연 · 진행 지연 · 미착수 자동 분류
- ⚠️ **선후행 충돌 탐지** — 선행 미완료 상태에서 후행 착수된 CPM 오류 감지
- 🤖 **만회 공정 AI 분석** — 지연 원인·만회 방안 제안
  - 서버 모드: 관리자 탭에 등록한 **서버 API 키**로 동작(직원 포함 전원 사용)
  - Pro(로컬): 개인 API 키 입력 — **키 접두어로 공급자 자동 인식** (`gsk_`=Groq, `sk-ant`=Claude, `sk-`=OpenAI)

### 멀티 현장 · 권한 (서버 모드)
- **현장(공사)별 데이터 분리** — 현장 선택 후 작업
- **역할 기반 권한** — 마스터 / 담당관리자 / 직원
  - 마스터: 전 현장 + 계정·현장 생성·수정·삭제 + 백업/복원
  - 담당관리자: 배정된 현장 관리 + 직원 추가 + 백업/복원
  - 직원: 배정 현장 보기·편집·공정회의 (백업/복원 불가)
- **관리자 탭 접근 코드** — 계정별 코드(마스터가 기본값 지정, 본인이 변경)
- 상단에 **공사명(현장명)** 표시, **🌐 서버 / 💻 로컬** 모드 배지

### 하도사 파일 · 공정회의
- **현장별 파일함** — `하도업체/<현장>/접수`·`보관` (전 구성원 업로드 가능)
- **취합 탭 2단 파일함** — 드래그&드롭 업로드 → 접수, 체크 후 ▶/◀로 접수↔보관 이동, 선택 다운로드(최신본)
- **공정회의(통합 비교)** — 접수 파일 기준 업체 선택(접수=활성/보관=비활성) → 한 타임라인에 적층 비교 → AI 간섭분석 → **[배포]**(참여사 최신 공정표를 보관으로, 접수에서 제거). **직원도 사용 가능**

### 별도 창 (로컬 대조/작성)
- 서버 접속 중 **🗔 로컬 파일(새 창)** / **🗔 새 공정표(새 창)** — 별도 윈도우에서 로컬 전용 편집
- 원도급 공정표와 배포본을 **두 창으로 나란히** 대조하며 수정

### 백업 / 복원 (서버 모드)
- **백업 위치**: `백업/<현장>/` (현장 생성 시 자동 생성, 현장명 변경 시 폴더명도 변경)
- **백업 대상**: 원도급 주 공정표. **서버 + 내 PC 동시 저장**
- **자동 백업**: 복원 직전 / 매일 22:00(+07:00 누락 시) — **직전 백업과 내용이 같으면 로그만**
- **복원**: 백업 목록에서 선택 → 복원 직전 자동백업 후 교체(전원 자동 새로고침)
- **이력**: `백업로그.jsonl` (누가·언제·백업/복원/삭제)
- **삭제**: 서버 PC(localhost)에서만 가능
- 🛡️ **데이터 보호 가드** — 기존 공종이 있는데 **빈 데이터로 덮어쓰기 시도 차단** + 보호 스냅샷

### JSON 저장/열기 (Pro · 단독)
- 파일 저장(다운로드) / 열기. 파일명 포맷 `MasterSchedule백업_<회사명>_<날짜>.json`
- 파일은 **만든 사람과 무관하게 portable** — 다른 사람이 열어도 정상

## 🚀 빠른 시작

### 1. 단독 실행 (Node.js 없이)
`master_schedule_v62.html` 을 브라우저로 바로 열어 Pro(로컬)로 사용. 데이터는 JSON 저장/열기로 관리.

### 2. 서버 연동 실행 (권장)
```bash
npm install
node server.js
```
브라우저에서 `http://localhost:3000` 접속.
- 엔터프라이즈 버전: 관리자 탭 → API 키(Groq 무료 / Claude) 등록 시 AI 분석 사용.
- 프로 버전: 본인 계정 → API 키(Groq 무료 / Claude) 등록 시 AI 분석 사용.

### 3. 자동 시작 등록 (Windows)
```
자동시작_설치.bat  실행
```
